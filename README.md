# 📚 Análise de Redes e Sistema de Recomendação em Comunidades de Leitura

## 🎯 Objetivo
Este projeto tem como objetivo analisar padrões de comportamento em uma rede de leitores e desenvolver um sistema de recomendação de livros com base na similaridade entre usuários.

A proposta simula uma rede social onde usuários são conectados a partir de preferências literárias, permitindo a identificação de comunidades e recomendações personalizadas.

---

## 🧠 Metodologia

O projeto foi desenvolvido em Python e dividido nas seguintes etapas:

- Criação de um dataset simulado contendo usuários, gêneros literários, livros e avaliações  
- Construção de uma rede (grafo) utilizando NetworkX  
- Cálculo de similaridade entre usuários com base em preferências e avaliações  
- Implementação de um sistema de recomendação  
- Análise exploratória dos dados  

---

## 📊 Principais Análises

### 🔗 Rede de Usuários
Usuários foram conectados com base na similaridade de preferências literárias, formando uma estrutura de grafo.

### ⭐ Centralidade
Identificação dos usuários mais influentes na rede através da centralidade de grau.

### 🧩 Comunidades
Detecção de grupos de usuários com interesses semelhantes.

### 🤝 Similaridade
Cálculo de score de similaridade entre usuários com base em gênero e avaliação.

### 📚 Recomendação de Livros
Sistema que sugere livros com base no comportamento de usuários semelhantes.
